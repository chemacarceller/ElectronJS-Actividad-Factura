# ElectronJS-Actividad-Factura

Desarrollo por fases de una actividad de gestión de facturas.<br />
Fase 1 sin persistencia de datos (varias versiones) <br />
Fase 2 con persistencia de datos en ficheros (versiones con modulos javascript import y modulos NodeJS require y version basada en Modelo-Vista-Controlador.<br />
Fase 3 con persistencia de datos en ficheros, MVC e implementación de canales de comunicación IPC e incorporación de ventanas adicionales, menus y cajas de dialogo nativas.<br />
Fase 4 con persistencia de datos en BBDD relacionales (MySQL y SQLite) y No-SQL (MongoDB) con IPC, ventanas adicionales, menus y caja de dialogo nativas + Modelo Vista Controlador Extendido (con clases Services y clases Repositories), así como la utilización del modulo propio mongoTool en la versión No-SQL<br />
Fase 5 : Versión análoga a la Fase 4 pero añadiendo un API de acceso a datos
